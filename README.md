# 📄AI Resume Screening System

## Overview
This project evaluates resumes based on a job description using AI.

## Features
- Skill Extraction (LLM)
- Matching + Scoring (Python logic)
- Explanation (LLM)
- LangSmith tracing

## Tech Stack
- Python
- LangChain
- Groq API

## How to Run
1. Clone repo
2. Install dependencies:
   pip install -r requirements.txt
3. Add .env file
4. Run:
   python main.py
